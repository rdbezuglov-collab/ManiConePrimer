\# 💅 Бот записи на маникюр



Telegram-бот для записи клиентов на маникюр.



\## Возможности

\- Проверка подписки на канал

\- Просмотр услуг и прайса с фото

\- Запись на удобное время (календарь + свободные слоты)

\- Отмена записи

\- Админ-панель для управления записями и слотами



\## Технологии

\- Python + pyTelegramBotAPI

\- SQLite база данных

\- dotenv для переменных окружения
